# Rocket.Chat Apps Experiments (rc-experiment)

This repository contains 10+ hands-on experiments created to understand how Google's `gemini-cli` behaves when generating Rocket.Chat Apps from plain English prompts. These experiments serve as proof-of-work validations for a GSoC proposal, identifying real limitations of AI app generation to directly influence the system design.

A correlative working POC implementation of the proposed system (which includes a `GEMINI.md` control layer, feature-specific skills, and scaffolding integrations) can be found at: [Khizarshah01/rc-geminicli](https://github.com/Khizarshah01/rc-geminicli).

---

### What We Are Doing Here
We utilize this directory as a sandbox environment to document these generated test applications (`testApp` instances, bots, etc.), showcase successes, detect AI-driven failure cases, and validate architectural decisions.
